# MusicNews - 音乐教育选题推荐系统

## 项目简介

MusicNews 是一个智能音乐教育选题推荐系统，专为音乐教育公众号提供每日选题灵感。

**核心亮点：时事热点 × 音乐教育 = 爆款选题**

## 功能特点

- 🔥 **热点抓取**：自动抓取微博、百度、知乎等平台热搜
- 🎹 **智能分析**：结合行业内账号的历史文章数据，分析爆款特征
- 🤖 **AI 生成**：基于热点话题，生成音乐教育选题建议
- 📧 **邮件推送**：每日定时发送选题到指定邮箱
- ⚙️ **自动运行**：生产环境由 Windows 后台服务每日自动执行，GitHub Actions 只保留手动应急触发

## 选题方向（5大类）

基于音乐教育行业头部公众号 877 篇历史文章分析：

1. **🎹 钢琴学习技巧类**（高频：练琴168次、技巧、方法34次）
2. **👨‍👩‍👧 家长教育指导类**（高频：孩子233次、家长57次）
3. **🎼 名家演奏赏析类**（高频：郎朗87次、演奏21次）
4. **📜 考级政策资讯类**（高频：考级75次）
5. **✨ 明星音乐故事类**（名人学琴经历）

## 选题策略

- **热点话题 + 音乐教育切入角度**
- **时事事件 + 钢琴学习关联**
- **名人动态 + 琴童家长启发**

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

编辑 `.env` 文件，配置：
- 邮件服务器信息
- AI API 密钥
- 代理设置（如需要）

### 3. 准备历史数据

将"头部公众号数据.xlsx"放入 `data/` 目录。

### 4. 运行生成

```bash
python main.py
```

选题邮件将自动发送到配置的邮箱。

## 技术栈

- **Python 3.8+**
- **requests**：HTTP 请求
- **pandas**：数据处理
- **OpenAI API**：AI 内容生成
- **smtplib**：邮件发送
- **GitHub Actions**：自动化部署
